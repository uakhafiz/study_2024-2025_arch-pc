## 2. Теоретическое введение

### 2.1 Арифметические операции в NASM

Язык ассемблера NASM предоставляет набор инструкций для выполнения арифметических операций над данными, хранящимися в регистрах и памяти.

**Особенности арифметических операций в NASM:**
- Операнды могут находиться в регистрах, памяти или быть непосредственными значениями
- Размер операндов должен быть согласован (байт, слово, двойное слово)
- Результат операций влияет на флаги процессора (ZF, CF, OF, SF)

### 2.2 Основные арифметические команды

**Сложение и вычитание:**

| Команда | Описание |
|---------|----------|
| `add op1, op2` | Сложение двух операндов, результат в op1 |
| `sub op1, op2` | Вычитание op2 из op1, результат в op1 |
| `inc op` | Увеличение операнда на 1 |
| `dec op` | Уменьшение операнда на 1 |
| `neg op` | Изменение знака операнда |

**Умножение и деление:**

| Команда | Описание |
|---------|----------|
| `mul op` | Беззнаковое умножение |
| `imul op` | Знаковое умножение |
| `div op` | Беззнаковое деление |
| `idiv op` | Знаковое деление |

**Особенности умножения и деления:**
- Для операций умножения и деления используются фиксированные пары регистров
- Результат умножения помещается в EDX:EAX для 32-битных операций
- При делении делимое должно находиться в определенных регистрах в зависимости от размера делителя

### 2.3 Преобразование данных

При работе с вводом-выводом в NASM важно различать:
- **Символьные данные** - ASCII коды символов
- **Числовые данные** - двоичное представление чисел
- Преобразование между форматами с помощью функций `atoi` (ASCII to integer) и `iprint` (integer print)

3.5 Программа lab6-3.asm - вычисление арифметических выражений

Задача: Реализация вычисления составного арифметического выражения с использованием нескольких операций.

Выражение: f(x) = (5 × 2 + 3) / 3

Реализация:

```asm
mov eax,5        ; EAX = 5
mov ebx,2        ; EBX = 2
mul ebx          ; EAX = 5 × 2 = 10
add eax,3        ; EAX = 10 + 3 = 13
xor edx,edx      ; Обнуление EDX для деления
mov ebx,3        ; EBX = 3
div ebx          ; EAX = 13 ÷ 3 = 4, EDX = 1
```

Результат:

```bash
@uakhafiz → ~/work/arch-pc/lab06$ ./lab6-3
Результат: 4
Остаток от деления: 1
```

Модификация для выражения (4 × 6 + 2) / 5:

```asm
mov eax,4        ; EAX = 4
mov ebx,6        ; EBX = 6
mul ebx          ; EAX = 4 × 6 = 24
add eax,2        ; EAX = 24 + 2 = 26
xor edx,edx      ; Обнуление EDX
mov ebx,5        ; EBX = 5
div ebx          ; EAX = 26 ÷ 5 = 5, EDX = 1
```

Результат:

```bash
@uakhafiz → ~/work/arch-pc/lab06$ ./lab6-3
Результат: 5
Остаток от деления: 1
```

Анализ: Программа демонстрирует правильную последовательность арифметических операций и корректную работу с остатком от деления.

4. Выполнение самостоятельной работы

4.1 Постановка задачи

Реализация программы для вычисления функции f(x) = (10 + 2x) / 3 с тестированием для значений x1 = 1 и x2 = 10.

4.2 Математические вычисления

x Вычисление Результат
1 (10 + 2×1) / 3 = 12 / 3 4
10 (10 + 2×10) / 3 = 30 / 3 10

4.3 Реализация программы

```asm
; variant.asm
; Программа для вычисления f(x) = (10 + 2x) / 3

%include 'in_out.asm'

SECTION .data
    msg_input: DB 'Введите значение x: ',0
    msg_result: DB 'Результат: ',0

SECTION .bss
    buf: RESB 80

SECTION .text
    GLOBAL _start

_start:
    ; Вывод приглашения для ввода x
    mov eax, msg_input
    call sprint

    ; Чтение введенной строки
    mov ecx, buf
    mov edx, 80
    call sread

    ; Преобразование строки в число
    mov eax, buf
    call atoi

    ; Вычисление f(x) = (10 + 2x) / 3
    mov ebx, 2
    mul ebx          ; EAX = 2 * x
    add eax, 10      ; EAX = 10 + 2x
    xor edx, edx     ; Обнуление EDX
    mov ebx, 3
    div ebx          ; EAX = (10 + 2x) / 3

    ; Вывод результата
    mov ebx, eax     ; Сохраняем результат
    mov eax, msg_result
    call sprint
    mov eax, ebx
    call iprintLF

    call quit
```
