# Лабораторная работа №3
# Язык разметки Markdown

## Цель работы

Освоение процедуры оформления отчетов с помощью легковесного языка разметки Markdown.

## Теоретическое введение

### Базовые сведения о Markdown

Markdown — это легковесный язык разметки, созданный для форматирования текста с минимальными усилиями.

**Основные элементы:**

| Элемент | Синтаксис | Пример |
|---------|-----------|--------|
| Заголовок | # H1, ## H2, ### H3 | # Заголовок 1 |
| Полужирный | **текст** | **жирный текст** |
| Курсив | *текст* | *курсив* |
| Ссылка | [текст](url) | [GitHub](https://github.com) |

### Списки

**Неупорядоченный список:**
* Элемент 1
* Элемент 2
  * Подэлемент 2.1
  * Подэлемент 2.2

**Упорядоченный список:**
1. Первый элемент
2. Второй элемент
   1. Подэлемент 2.1
   2. Подэлемент 2.2

### Математические формулы

Внутритекстовые формулы: $\sin^2(x) + \cos^2(x) = 1$

Выключные формулы:
$$\sin^2(x) + \cos^2(x) = 1$$
