## 2. Теоретическое введение

### 2.1 Базовые сведения о Markdown

Markdown — это легковесный язык разметки, созданный для удобного форматирования текста с возможностью преобразования в HTML, PDF и другие форматы.

**Основные элементы Markdown:**

| Элемент | Синтаксис | Пример |
|---------|-----------|--------|
| Заголовки | `#`, `##`, `###` | `# Заголовок 1` |
| Полужирный текст | `**текст**` | `**bold**` |
| Курсивный текст | `*текст*` | `*italic*` |
| Полужирный курсив | `***текст***` | `***bold italic***` |

### 2.2 Оформление списков

**Упорядоченный список:**
```markdown
1. Первый пункт
2. Второй пункт
   1. Подпункт
   2. Подпункт
3. Третий пункт
```

Неупорядоченный список:

```markdown
* Пункт 1
* Пункт 2
  * Подпункт A
  * Подпункт B
* Пункт 3
```

2.3 Оформление ссылок и изображений

Ссылка:

```markdown
[Текст ссылки](https://example.com "Подсказка")
```

Изображение:

```markdown
![Подпись к рисунку](image.png "Подсказка"){#fig:fig1 width=50%}
```

2.4 Оформление формул

В Markdown математические формулы оформляются с использованием синтаксиса LaTeX:

Встроенная формула: $\sin^2(x) + \cos^2(x) = 1$ → $\sin^2(x) + \cos^2(x) = 1$

Выключенная формула с меткой:

```markdown
$$
\sin^2(x) + \cos^2(x) = 1
$$ {#eq:eq1}
```

6. Контрольные вопросы для самопроверки

1. Что такое Markdown?

Markdown — это легковесный язык разметки, разработанный для форматирования текста с использованием простого синтаксиса, который легко читается и может быть преобразован в HTML, PDF и другие форматы.

2. Как в Markdown задается начертание шрифтов?

Начертание Синтаксис Пример
Полужирный **текст** или __текст__ bold
Курсивный *текст* или _текст_ italic
Полужирный курсив ***текст*** bold italic

3. Как в Markdown оформляются списки?

Упорядоченный (нумерованный) список:

```markdown
1. Первый пункт
2. Второй пункт
   1. Подпункт
   2. Подпункт
```

Неупорядоченный (маркированный) список:

```markdown
* Пункт 1
* Пункт 2
  * Подпункт A
  * Подпункт B
```

4. Как в Markdown оформляются изображения и ссылки на них?

Вставка изображения:

```markdown
![Подпись к рисунку](image.png "Подсказка"){#fig:fig1 width=50%}
```

Ссылка на изображение в тексте:

```markdown
(рис. [-@fig:fig1])
```

5. Как в Markdown оформляются математические формулы и ссылки на них?

Встроенная формула:

```markdown
$\sin^2(x) + \cos^2(x) = 1$
```

Выключенная формула:

```markdown
$$
\sin^2(x) + \cos^2(x) = 1
$$ {#eq:eq1}
```

Ссылка на формулу:

```markdown
Смотри формулу ([-@eq:eq1])
```
